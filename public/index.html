Pineapple Gallery

A modern web application for managing and displaying beautiful image galleries. Built with Hono and powered by Cloudflare Workers, D1, and R2 for optimal performance.

Features

✨ User-Friendly Interface
🌐 Cloudflare Integration
🌍 Multi-Language Support
🔒 Secure Admin Access

Tech Stack

  • Frontend: Hono (SSR), HTMX
  • Backend: Cloudflare Workers
  • Storage: Cloudflare D1 & R2
  • Cache: Cloudflare KV
